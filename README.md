使用docker搭建ldap服务,在redmine配置ldap服务流程，包括ldap客户端整合到一起了！！！通过图片学习
使用镜像

```
#导入镜像
docker pull muen/openldap:develop-dev
或者官方
docker pull osixia/openldap:develop-dev
```
